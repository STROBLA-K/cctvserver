Power Automate Integration Guide

When the server is live (e.g., on a local network or Heroku), use the following steps:

  1. Use the HTTP action.
  2. Set Method to GET.
  3. Set URI to http://YOUR_SERVER_IP:5000/snapshot
  4. The response will contain the image data ready for saving or processing.
